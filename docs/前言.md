# 前言

欢迎来到安徽理工大学生存指南！

本指南旨在为安徽理工大学的同学们提供实用的校园生活、学习、升学等方面的经验和建议。

## 指南的目的

本指南由安徽理工大学的同学们共同编写，旨在：

- 帮助新生快速适应大学生活
- 分享学习经验和技巧
- 提供升学（保研、考研）相关信息
- 记录校园生活的点点滴滴

## 如何使用本指南

本指南按照不同的主题分为多个章节，你可以：

- 按需查找感兴趣的内容
- 使用搜索功能快速定位信息
- 参考学长学姐的经验分享

## 贡献

本指南是一个开源项目，欢迎所有安徽理工大学的同学贡献内容。如果你有好的建议或经验分享，请查看 [贡献指南](./贡献指南.md)。
