# 贡献指南

感谢你对安徽理工大学生存指南的关注！我们欢迎所有同学贡献内容。

## 如何贡献

### 方式一：通过 GitHub Pull Request

1. Fork 本仓库
2. 创建你的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交你的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 打开一个 Pull Request

### 方式二：直接编辑

如果你不熟悉 Git，也可以：

1. 在 GitHub 上找到要编辑的文件
2. 点击右上角的编辑按钮（铅笔图标）
3. 进行修改并提交

## 内容规范

### 写作风格

- 使用简体中文
- 保持客观、实用的态度
- 避免过于主观或偏激的观点
- 提供具体、可操作的建议

### 格式要求

- 使用 Markdown 格式
- 标题层级清晰
- 适当使用列表、代码块等格式
- 添加必要的链接和引用

## 内容分类

你可以贡献以下类型的内容：

- **学习经验**：课程学习、考试技巧、学习方法等
- **校园生活**：住宿、饮食、交通、社团等
- **升学信息**：保研、考研、出国等
- **实习就业**：实习经验、求职技巧等
- **其他**：任何对同学们有帮助的内容

## 提 Issue

如果你发现：

- 内容有误或需要更新
- 有新的主题需要添加
- 对指南有改进建议

欢迎在 GitHub 上提 Issue 讨论。

感谢所有为本指南做出贡献的同学们！
