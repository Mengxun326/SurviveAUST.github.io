# SurviveAUST

安徽理工大学生存指南。

## 本地部署与调试

安装 Node.js/Git。

```bash
git clone https://github.com/Mengxun326/SurviveAUST.github.io.git
cd SurviveAUST.github.io
npm install
npm run dev
```

访问 http://localhost:5173 查看本地预览。

## 编译构建

```bash
npm run build
```

构建产物在 `.vitepress/dist` 目录。

## 编译 PDF

```bash
npm run export-pdf
```

## 贡献指南

欢迎所有安徽理工大学的同学贡献内容！请查看 [贡献指南](./docs/贡献指南.md) 了解如何参与。
